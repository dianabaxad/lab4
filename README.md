# Lab Generator - Генераторы и сортировка данных
# Описание проекта
Lab Generator - это десктопное приложение на Python с графическим интерфейсом PySide6 для решения трех задач, связанных с генераторами и обработкой данных. Приложение демонстрирует работу с функциями-генераторами, параллельными вычислениями и сортировкой данных.

# Задание 1: Генератор сочетаний из трех цифр
Генератор всех возможных комбинаций из трех цифр (000-999)
Преобразование в целые числа
Параллельная версия для ускорения генерации
Вывод первых N значений с форматированием

# Задание 2: Генератор значений функции
Гибкий генератор значений функций
Поддержка любых пользовательских функций
Настройка интервала и шага
Параллельное вычисление значений

# Задание 3: Сортировка словаря
Сортировка словарей по убыванию ключей
Поддержка различных форматов ввода
Встроенные примеры словарей
Параллельная сортировка больших словарей

# Интерфейс
Вкладки для каждой задачи
Поля ввода диапазонов, количества значений и строки чисел
Кнопки запуска генераторов и параллельных версий
Область вывода результатов для удобного просмотра

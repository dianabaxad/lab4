# Lab Generator - Генераторы и сортировка данных
# Описание проекта
Lab Generator - это десктопное приложение на Python с графическим интерфейсом PySide6 для решения трех задач, связанных с генераторами и обработкой данных. Приложение демонстрирует работу с функциями-генераторами, параллельными вычислениями и сортировкой данных.

# Задание 1: Генератор сочетаний из трех цифр
1. Введите количество значений для вывода (по умолчанию 50)
2. Нажмите "Сгенерировать" для последовательной версии
3. Результаты отобразятся в текстовом поле

# Задание 2: Генератор значений функции
1. Настройте параметры:
    - Начало интервала (a): -20
    - Конец интервала (b): 100
    - Функция f(x): -1.5*x + 2
    - Количество значений: 20
2. Нажмите "Вычислить" для последовательного расчета

# Задание 3: Сортировка словаря
1. Нажмите "Отсортировать" для выполнения операции
2. Результаты отобразятся в виде отсортированного списка

# Интерфейс
1. Вкладки для каждой задачи
2. Поля ввода диапазонов, количества значений и строки чисел
3. Кнопки запуска генераторов и параллельных версий
4. Область вывода результатов для удобного просмотра

## Технические требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- PySide6 (для GUI интерфейса)

## Инструкции по запуску
-Скачайте или склонируйте файлы проекта
-Откройте терминал в папке с проектом
-Запустите главный файл приложения:
```bash
python Main.py
```
Для запуска pytest:
```bash
python pytests.py
```
